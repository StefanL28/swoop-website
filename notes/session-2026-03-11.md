# Session — March 11, 2026

## What We Did

Built the complete marketing landing page for **Swoop** — a chat-first discount shopping platform available in Abidjan, Ivory Coast.

---

## Tech Stack

| Layer | Choice |
|---|---|
| Framework | Astro 5 (static output) |
| Styling | Tailwind CSS v4 |
| Animations | GSAP + ScrollTrigger |
| Font | Plus Jakarta Sans (Google Fonts) |
| Deployment target | Hostinger `public_html` |

---

## Project Structure

```
src/
├── layouts/Layout.astro        # Base HTML, meta tags, Google Fonts
├── pages/index.astro           # Single page composing all sections
├── components/
│   ├── Navbar.astro            # Sticky nav + full-screen mobile menu overlay
│   ├── Hero.astro              # Two-column hero with CSS phone mockup
│   ├── SocialProof.astro       # Stats bar with scroll-triggered count-up
│   ├── FeatureSection.astro    # Reusable alternating phone+text block
│   ├── BentoGrid.astro         # 6-card secondary features grid
│   ├── BentoCard.astro         # Individual bento card
│   ├── HowItWorks.astro        # 3-step process with connecting line
│   ├── Testimonials.astro      # 3 testimonial cards (horizontal scroll on mobile)
│   ├── TestimonialCard.astro   # Individual testimonial card
│   ├── AvatarStack.astro       # Overlapping avatar stack
│   ├── FAQ.astro               # FAQ accordion (vanilla JS)
│   ├── FAQItem.astro           # Individual FAQ item
│   ├── FinalCTA.astro          # Dark CTA section with app store buttons
│   ├── Footer.astro            # 3-column footer
│   └── ui/
│       ├── Button.astro        # Reusable pill button (multiple variants)
│       └── SectionHeading.astro # Reusable section heading
├── scripts/animations.ts       # All GSAP ScrollTrigger animations
├── styles/global.css           # Tailwind + CSS custom properties (color palette)
└── data/
    ├── features.ts             # Main features + bento grid data
    ├── testimonials.ts         # 3 testimonials
    └── faq.ts                  # 6 FAQ items
```

---

## Page Sections (top to bottom)

1. **Navbar** — Logo + nav links + CTA pill; transparent → white + shadow on scroll; hamburger mobile with full-screen overlay
2. **Hero** — "🇨🇮 Disponible à Abidjan" badge, headline with orange accent words, subtitle, CTA button, avatar stack, CSS iPhone mockup with chat UI, floating "-50%" discount card, floating star badge
3. **Social Proof** — "10,000+ Users | 500+ Offers | 50+ Partners" with scroll-triggered count-up animation
4. **Feature Sections** — 3 alternating phone+text blocks with decorative SVG blobs (chat, personalized deals, local map)
5. **Bento Grid** — 6 cards: Instant Notifications, Simple UI, Free, Verified Partners, Daily Offers, Secure
6. **How It Works** — 3 numbered steps with desktop connecting line
7. **Testimonials** — 3 cards with star ratings, author name + Abidjan neighborhood; horizontal scroll on mobile
8. **FAQ** — 6 items with smooth JS accordion (expand/collapse, +/× icon rotation)
9. **Final CTA** — Dark background, headline, App Store + Google Play buttons
10. **Footer** — Logo, tagline, quick links, contact, social icons, copyright
